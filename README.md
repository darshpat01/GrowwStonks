Groww Frontend Assignment

Features:
* Creative UI
* NextJs Server Side rendering implemented.
* Follow a standard well defined folder structure.
* Responsive UI
* Used Ant Design Charts for line graph
* Cached API responses
* Light and Dark Mode Themes
* Deployed on vercel
